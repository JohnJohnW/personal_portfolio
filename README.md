# John Wynter Portfolio

This is the source code for my personal portfolio built with React. It showcases my projects, articles and contact details in a single page application.

## Getting started

Install dependencies and start the development server:

```bash
npm install
npm start
```

The site will be available at `http://localhost:3000`.

## Building

To create a production build:

```bash
npm run build
```

## Testing

Run unit tests with:

```bash
npm test
```
